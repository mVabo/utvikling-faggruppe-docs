# Basics

## Cloning a remote repository

To get the project on your machine locally enter:

```bash
~$ git clone <url-here>
```

:::tip Credentials

When cloning the repository in DevOps there is a button with the text "Generate Git Credentials", get your password from here, dont use your normal AD password

:::

## Pusing local changes

There is three steps to pushing code to a remote repository

1. Stage all changes

```bash
~$ git add .
```

2. Commit staged changes

```bash
~$ git commit -m "yolo commit"
```

3. Push changes to remote

```bash
~$ git push
```
